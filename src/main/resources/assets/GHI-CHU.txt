Thu muc nay de chua anh giao dien cua launcher (chua tao san vi
chua co file that):

- background.png  - anh nen cua so launcher
- logo.png        - logo hien o goc tren ben trai
- icon.ico        - icon cho file .exe / icon cua so
- play.png        - icon nut PLAY (neu muon dung anh thay vi chu)
- loading.png     - anh hien khi dang tai / dang khoi chay game

Khi co file that, bo vao thu muc nay (src/main/resources/assets/),
Gradle se tu dong dong goi chung vao ung dung. Vi du noi logo.png
vao thay cho chu "ME LAUNCHER" trong launcher.fxml:

  <ImageView fitHeight="28">
      <image><Image url="@../assets/logo.png"/></image>
  </ImageView>

Hoac dat anh nen bang CSS trong dark.css:

  .root-pane {
      -fx-background-image: url("/assets/background.png");
      -fx-background-size: cover;
  }
